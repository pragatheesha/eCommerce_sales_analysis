# E-commerce Sales Analysis

# Project Overview
This project analyzes e-commerce sales data using Python. The analysis includes revenue calculation and visual insights using charts.

# Setup Instructions
1. Install Python
2. Install dependencies:
pip install -r requirements.txt
3. Run the project:
python main.py

# Folder Structure
- data/ → dataset
- visualizations/ → charts
- report/ → analysis report
- main.py → main analysis script

# Charts Used
- Bar chart (Sales by Product)
- Pie chart (Revenue by Product)

# Tools Used
- Pandas
- Matplotlib

## Outcome
This project demonstrates a complete data analysis pipeline including loading, analyzing, visualizing, and reporting data.
